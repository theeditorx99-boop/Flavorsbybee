# FlavorsByBee

[![Netlify Status](https://api.netlify.com/api/v1/badges/008c10e1-d194-4199-8883-156575b1fdba/deploy-status)](https://app.netlify.com/projects/roya1cocoa/deploys)

Welcome to the FlavorsByBee source code repository! This is a modern, high-performance web application built with React, TanStack Start, and Tailwind CSS.

## Overview
FlavorsByBee is a boutique chocolaterie dedicated to crafting unforgettable tasting experiences through small-batch chocolates, luxurious truffles, and elegant gifting collections. 

## Tech Stack
- **Framework:** React + TanStack Router / Start
- **Styling:** Tailwind CSS + Vanilla CSS
- **Animations:** Framer Motion
- **Deployment:** Netlify

## Local Development
To run this project locally:

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
